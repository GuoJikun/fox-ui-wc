# fox-ui-wc

符合 Web Components 规范的 web 组件库

## 使用须知

```
-   所有组件的使用方式都和原生方式一致（获取元素的属性...）
-   自定义事件（非原生事件）都需要使用`addEventListener`来绑定
```
